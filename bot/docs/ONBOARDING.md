# 🎉 新規ユーザーのチュートリアル体験フロー

## 📖 概要
新規ユーザーがDiscordサーバーに参加してから、みやにゃんのチュートリアルシステムを通じてサーバーの使い方を習得するまでの完全な体験フローを説明します。

---

## 🚪 Phase 1: サーバー参加時（自動発動）

### 1. サーバー参加の瞬間
新規ユーザーがサーバーに参加すると、**みやにゃんが自動的に検出**して歓迎メッセージを送信します。

### 2. ウェルカムメッセージ（パブリックチャンネル）
```

@新規ユーザー さん、[サーバー名]へようこそですにゃ〜！

🐈 みやにゃんと申しますにゃ！
私がこのサーバーの使い方をご案内させていただきますにゃ〜
技術的なことやコミュニティのサポートが得意ですにゃ！

📚 チュートリアルを始めませんか？
このサーバーを最大限活用できるように、
ステップバイステップでご案内しますにゃ〜

下の 🎓 ボタンを押してチュートリアルを開始してくださいにゃ！

困ったことがあったら「@みやにゃん ヘルプ」と呼んでくださいにゃ〜
```

### 3. DMでの個人的なご挨拶
```
🐈 みやにゃんからのご挨拶

```

## 🎮 Phase 4: インタラクティブな進行

### 利用可能なコマンド
ユーザーは各ステップで以下のコマンドを使用できます：

| コマンド | 動作 |
|---------|------|
| `次へ` / `できた` / `ok` | 次のステップに進む |
| `スキップ` / `skip` | 現在のステップをスキップ |
| `ヘルプ` / `help` | ヘルプメッセージを表示 |
| `終了` / `quit` | チュートリアルを終了 |
| `一時停止` / `pause` | チュートリアルを一時停止 |

### ヘルプメッセージ例
```
🆘 チュートリアルヘルプ

チュートリアルでお困りですか？みやにゃんがお手伝いしますにゃ〜

📝 使えるコマンド
• `次へ` / `できた` - 次のステップに進む
• `スキップ` - 現在のステップをスキップ
• `ヘルプ` - このヘルプを表示
• `終了` - チュートリアルを終了

❓ よくある質問
Q: チャンネルが見つからない
A: サーバーによってチャンネル名が違う場合があります。似た名前のチャンネルを探してみてくださいにゃ

Q: 権限がない
A: 一部の機能は時間が経ってから使えるようになりますにゃ〜
```
